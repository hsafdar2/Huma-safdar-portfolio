# Password Management System

## Overview
This project is a Python-based password management system that allows users to store and manage login information. It demonstrates the use of user input, conditionals, and loops.

## Skills Used
- Python
- Loops
- Conditionals
- User Input Handling

## What I Learned
This project helped me understand how to structure a program, manage user data, and apply logic to real-world scenarios.

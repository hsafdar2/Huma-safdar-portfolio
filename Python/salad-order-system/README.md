# Salad Order System

## Overview
This project is a Python program that simulates a simple salad ordering system. It takes user input and uses program logic to process choices and display results.

## Skills Used
- Python
- User Input
- Conditional Statements
- Basic Program Logic

## What I Learned
This project helped me practice building interactive Python programs and organizing user choices using clear logic.

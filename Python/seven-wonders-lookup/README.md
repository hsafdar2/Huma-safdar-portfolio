# Seven Wonders Lookup

## Overview
This project is a Python program that allows users to look up information about the Seven Wonders. It demonstrates the use of data structures and user interaction.

## Skills Used
- Python
- User Input
- Conditional Logic
- Basic Data Handling

## What I Learned
This project helped me understand how to organize and retrieve information using Python, as well as how to interact with users through input and output.
